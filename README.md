# TikTok Gift Proxy Server

Промежуточный сервер для получения подарков TikTok LIVE.

## Деплой на Render.com (бесплатно)

1. Зайди на render.com → Sign Up (бесплатно)
2. New → Web Service
3. Connect GitHub → выбери этот репозиторий
4. Настройки автоматически подтянутся из render.yaml
5. Нажми Deploy
6. Скопируй URL вида: https://tiktok-gift-proxy.onrender.com
